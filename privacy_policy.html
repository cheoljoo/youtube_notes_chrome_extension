개인정보 보호 정책

YouTube Notes 확장 프로그램(이하 "확장")의 개인정보 보호 정책입니다. 당사는 사용자의 개인정보 보호를 매우 중요하게 생각합니다.

1. 수집하는 정보

이 확장은 다음 정보만을 수집합니다:

  • YouTube 페이지 메타데이터: 제목, 게시 시간, URL
  • 사용자 입력: 태그, 의견 (사용자가 수동 입력)
  • 타임스탬프: 노트 저장 시간

수집하지 않는 정보:

  • 브라우징 이력
  • 개인 식별 정보
  • 쿠키 또는 추적 데이터
  • 위치 정보

2. 데이터 저장 방식

모든 데이터는 로컬 장치에만 저장됩니다:

  • Chrome 로컬 스토리지: 노트 데이터 (chrome.storage.local)
  • Chrome 동기 스토리지: 태그 목록 (chrome.storage.sync - Chrome 계정 동기화)

외부 서버로 데이터를 전송하지 않습니다.

3. 제3자 공유

이 확장은:

  • 사용자 데이터를 제3자와 공유하지 않습니다
  • 광고 네트워크나 분석 서비스를 사용하지 않습니다
  • 원격 서버로 데이터를 전송하지 않습니다

4. CSV 내보내기

사용자는 언제든지 "CSV 다운로드" 기능을 통해 자신의 모든 데이터를 내보낼 수 있습니다. 이는 완전히 사용자의 제어 하에 있습니다.

5. 확장 제거

이 확장을 제거하면 Chrome은 자동으로 모든 관련 데이터를 삭제합니다.

6. 권한 설명

  • storage: 사용자의 노트를 로컬에 저장
  • activeTab: 현재 YouTube 페이지의 메타데이터 접근
  • scripting: 페이지에서 제목 및 게시 시간 추출

7. 정책 변경

이 정책이 변경되면, 확장의 새 버전을 통해 공지됩니다.

8. 문의

개인정보 보호와 관련된 문의사항은 GitHub 이슈를 통해 연락주세요.

요약: YouTube Notes는 귀하의 데이터를 로컬에만 저장하며, 외부로 전송하거나 제3자와 공유하지 않습니다.

Personal DB (개인용 Supabase) 안내
Using your own free Supabase project gives you a personal DB, so even the free tier provides ample storage for reliable use.
How to proceed: Create your Supabase project, then enter Supabase URL and Supabase API Key (anon/public) in Settings, and set Manual user email / ID (required). See guides: SUPA_setup_for_personal.en.md (English)  |  SUPA_setup_for_personal.md (Korean)
